# TharMesh

**Offline Mesh Messaging for Android — No Internet Required**

TharMesh is a next-generation offline communication platform built for environments where internet connectivity is unreliable, unavailable, or intentionally restricted.

It enables direct device-to-device messaging using Bluetooth and WiFi Direct mesh networking — without relying on centralized servers.

---

## 🚀 Why TharMesh?

In many regions, stable internet infrastructure is not guaranteed.

TharMesh is designed to:

- 📡 Operate fully offline
- 🔗 Enable peer-to-peer device communication
- 🏘 Support village-scale mesh networks
- 🛟 Function during disasters and network shutdowns
- 🏛 Serve as infrastructure-grade communication backbone

---

## 🔥 Key Features

- Offline messaging (No mobile data required)
- Bluetooth + WiFi Direct mesh transport
- Custom-built networking layer
- Designed for scalability
- Android API 24+ support
- Lightweight architecture
